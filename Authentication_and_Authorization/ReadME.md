# Authentication and Authorization 

- ### Authentication : Authentication is the process of verifying a user's identity, usually using credentials such as an email and password, to confirm that the user is who they claim to be. ✅

- ### Authorization : Authorization is the process of checking what a user is allowed to do after they have logged in.

# Session and Cookies 

- ### Session : A session is a mechanism that allows a server to remember information about a user across multiple requests after they have been authenticated.

- ### Cookies : A cookie is a small piece of data stored in the user's browser that helps websites remember information across requests.

# what is Encryption ? 

- ### Encryption is the process of converting readable data into an unreadable format to protect it from unauthorized access.

# what is Decryption ? 

- ### Decryption is the process of converting encrypted data back into its original readable form using the correct key.

# what is Hash ? 

- ### Hashing is a one-way process that converts data into a fixed-length value called a hash, commonly used to securely store passwords.
